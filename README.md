# Torchlight Infinite Build Planner

Plan, create, and share your Torchlight Infinite builds. Choose your hero, customize skills, and optimize your equipment for maximum power.

## Features

- **11 Heroes** - Select from Rehan, Carino, Gemma, Moto, Erika, Youga, Thea, Iris, Bing, Rosa, and Selena
- **Hero Traits** - Each hero has multiple specialization traits
- **Skill Builder** - Create skill links with main skills and up to 5 support gems each
- **Equipment Panel** - Plan your gear with 12 equipment slots including Vorax Limbs
- **Build Sharing** - Save and share your builds with the community
- **User Profiles** - Create an account to save builds permanently

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Zustand (state management)
- React Router

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/laurent7850/torchlight-build-planner.git
cd torchlight-build-planner

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```
